# mental-health-chat-bot-
This repository contains two generations of Mental Health Chatbots developed using Python, demonstrating the evolution from traditional rule-based Natural Language Processing (NLP) techniques to modern Large Language Model (LLM) powered conversational AI.

The first chatbot utilizes NLTK, text preprocessing, tokenization, stopword removal, stemming, and a structured knowledge base to provide responses through a rule-based conversational system. The application also includes a graphical user interface built using Tkinter.

The second chatbot leverages the Llama 3.3 70B language model through API integration to enable context-aware conversations. It further performs AI-driven emotional analysis by evaluating user interactions and generating structured emotional profiles, which are visualized through interactive radar charts using Plotly.

Key Features:
• Rule-based NLP chatbot using NLTK
• Text preprocessing and keyword-based response matching
• Tkinter-based graphical user interface
• LLM-powered conversational AI using Llama 3.3
• API integration for real-time AI responses
• Emotional profile generation and sentiment-oriented analysis
• JSON-based analytics processing
• Interactive Plotly dashboard visualizations

Technologies Used:
• Python
• NLTK
• Pandas
• Tkinter
• Llama 3.3
• Groq API
• JSON
• Plotly

This project demonstrates practical applications of conversational AI, Natural Language Processing, Large Language Models, API integration, data visualization, and user-centered wellness technology.
